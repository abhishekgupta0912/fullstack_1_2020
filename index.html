About Course

Front-end web development is everything in programming the user interface of a web application. Typically it refer to the Hyper Text Markup Language(HTML). Cascading Style Sheet(CSS) and JavaScript portion of web site production as opposed to the database or srver-side programming. It encompasses everything from building a simple page of HTML text to creating complex, responsive HTML5 website designed to be accessed via various different browsers, devices and screen size.
